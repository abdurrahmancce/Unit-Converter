# 📏 Unit Converter

A modern and beginner-friendly Unit Converter application built with Python and Tkinter.

This desktop application allows users to convert Temperature, Length, and Weight units through an intuitive graphical interface. It includes real-time conversion, conversion history, unit swapping, and dark/light themes for an enhanced user experience.

---

## 🚀 Features

### 🌡 Temperature Conversion
Convert between:
- Celsius (°C)
- Fahrenheit (°F)
- Kelvin (K)

### 📏 Length Conversion
Convert between:
- Meter (m)
- Kilometer (km)
- Centimeter (cm)
- Millimeter (mm)
- Mile (mi)
- Inch (in)
- Foot (ft)

### ⚖ Weight Conversion
Convert between:
- Kilogram (kg)
- Gram (g)
- Pound (lb)
- Ounce (oz)

## 🖥 Usage

1. Launch the application.
2. Select a category:
   - Temperature
   - Length
   - Weight
3. Choose the source unit.
4. Enter a value.
5. Choose the target unit.
6. Click **Convert**.
7. View the result instantly.
8. Use **Swap Units** to reverse units.
9. Check the conversion history panel.
